# laravel-postcodes
Laravel Package, postcode utilities.

## Install
Update `composer.json`
```$xslt
{
    "repositories": [
        ...
        {
            "type": "vcs",
            "url": "https://github.com/robconvery/laravel-postcodes"
        }
    ],
    ...
    "require": [
        ...
        "robconvery/laravel-postcodes": "^1.0"
    ]
}
```
Run `composer update`
